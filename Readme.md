# prime

Primality test.

## Installation

``` shell
$ component install KenanY/prime
```

## API

### prime(val)

Check primality of a number `val`:

``` javascript
prime(13);
// => true
```

## Development

``` shell
# Build
$ make

# Run tests
$ npm install mocha should
$ make test
```
