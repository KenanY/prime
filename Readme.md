# prime

Primality test.

## Installation

``` shell
$ component install KenanY/prime
```

## API

### prime(val)

Check primality of a number `val`:

``` javascript
prime(13);
// => true
```

## Development

``` shell
$ npm install -g component
$ npm install mocha should
$ make
```

## Links

  - If you'd like a more powerful primality test, check out
